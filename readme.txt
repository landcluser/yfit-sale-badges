== Description ==
YFIT Sale Badges is a plugin that will transform your default WooCommerce sale badges into something that your customers can relate to and that will grab their attention instantly. Find the settings on the WooCommerce > Settings > YFIT Badges page of your WordPress Dashboard.

== Features ==
* Change the style according to your current collection, season or holiday using the awesome presets included in the plugin.
* The plugin seamlessly integrates itself into the WooCommerce interface.
* You can even set a different sale badge for each product individually!
* 3 Special badges are also included to which you can add your own custom text!
* Flexible customize label, title.
* Optimized speed.


== Installation ==
1. Upload the plugin files to the directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the "Plugins" screen in WordPress.
